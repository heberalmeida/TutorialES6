# ECMAScript 6 Tutorial | Tutorial ECMAScript 6

Vue 3 application with full internationalization (i18n) in **English** and **Portuguese (Brazil)**.

**Author:** Heber Almeida

**Demo:** https://heberalmeida.github.io/TutorialES6/

**Repository:** https://github.com/heberalmeida/TutorialES6

## Features

- **Vue 3** with Composition API
- **Vue Router** for navigation
- **Vue I18n** for English and Portuguese (BR)
- 32 chapters in both languages
- Dark theme with modern UI
- Responsive layout with sidebar
- Language switcher (EN / PT-BR)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

The app is configured for GitHub Pages at `https://heberalmeida.github.io/TutorialES6/`.

1. Push the code to the `main` or `master` branch
2. In the repo: **Settings** → **Pages** → **Source**: select **GitHub Actions**
3. Each push will trigger a deploy
